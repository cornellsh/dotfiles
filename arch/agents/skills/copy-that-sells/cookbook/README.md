# Cookbook: Worked Examples

Five end-to-end examples of the skill applied to real briefs. Each example shows the brief, the full output in the standard skill format, and a postmortem naming the frameworks and craft moves that did the work. Use the cookbook as a few-shot reference when stuck. Steal the structure, not the words.

## Index

1. [Billboard for a LatAm fintech](01-billboard-latam-fintech.md). Spanish, OOH, rioplatense voseo. Demonstrates compression in Spanish and the use of `spanish-craft.md`.
2. [B2B SaaS pricing page](02-saas-pricing-page.md). English, conversion structure. Demonstrates the pricing-page blueprint from `formats.md`.
3. [DTC manifesto](03-dtc-manifesto.md). English, voice-first writing. Demonstrates Voice 3 (manifesto) from `voice-bank.md`.
4. [Founder cold email](04-founder-cold-email.md). English, outbound. Demonstrates the cold-email structure from `formats.md` and the BFD framework from `frameworks.md`.
5. [App Store listing](05-app-store-listing.md). English, ASO + selling. Demonstrates compression at multiple character limits.

## How to use the cookbook

Read the example closest to your brief. Then read the postmortem. Then write your own. The cookbook is not a template library. It is proof that the workflow produces output the user can ship, when followed.
